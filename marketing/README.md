# Marketing Package

This folder contains marketing materials for AI Keyboard launch and promotion.

## Contents

- **TWITTER_LAUNCH_THREAD.md** - X/Twitter launch thread and announcement
- **REDDIT_POST.md** - Reddit-friendly launch post (r/Android)
- **DISCORD_ANNOUNCEMENT.md** - Discord announcement message
- **VIDEO_SCRIPT.md** - Promotional video script (30-45 seconds)
- **SOCIAL_BANNERS.md** - Text-only social media banner descriptions

## Usage

### X/Twitter

1. Use the launch thread for a multi-tweet announcement
2. Use the short announcement for a single tweet
3. Tag [@MjYoke1111](https://x.com/MjYoke1111) for mentions

### Reddit

1. Post in r/Android with the Reddit post template
2. Adjust tone and details as needed
3. Respond to comments and questions

### Discord

1. Post in relevant Discord servers (Android development, privacy, etc.)
2. Use the announcement template
3. Engage with community feedback

### Video

1. Use the video script to create a promotional video
2. Record screen demos of the app
3. Add visuals and music
4. Post on YouTube, X/Twitter, etc.

## Social Media Guidelines

- **Be Authentic**: Share genuine excitement about the project
- **Engage**: Respond to comments and questions
- **Be Respectful**: Maintain professional tone
- **Credit**: Credit contributors and open-source projects
- **Privacy-First**: Emphasize privacy and offline processing
