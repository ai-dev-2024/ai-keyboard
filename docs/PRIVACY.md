# Privacy Policy

**Last Updated**: November 2024

## Introduction

AI Keyboard is committed to protecting your privacy. This Privacy Policy explains how we handle your data when you use our keyboard application.

---

## Privacy-First Design

**AI Keyboard processes all voice recognition 100% offline on your device.** No audio data is ever sent to external servers. Your privacy is our top priority.

---

## Data Collection

**AI Keyboard does not collect, transmit, or store any personal data on external servers.**

### What We Do NOT Collect

- ❌ Voice/audio recordings
- ❌ Typing history
- ❌ Personal information
- ❌ Analytics or tracking data
- ❌ Device identifiers
- ❌ Location data
- ❌ Contact information

### What We Store Locally

All data is stored locally on your device in the app's private directory:

- ✅ **Personal dictionary**: Words you add are stored in a local Room database
- ✅ **Clipboard history**: Managed locally, never transmitted
- ✅ **Keyboard settings and preferences**: Stored locally using DataStore
- ✅ **AI models**: Downloaded models are stored in app-private storage

**All data remains on your device and is never transmitted to external servers.**

---

## Voice Input & Microphone

### Offline Processing

- **100% Offline**: All voice recognition happens on-device
- **No Cloud Processing**: No audio data is sent to cloud servers
- **User Control**: Voice input is only active when you use it
- **No Background Recording**: Microphone is only active during voice input sessions

### Microphone Permission

AI Keyboard requires the `RECORD_AUDIO` permission for voice input functionality:

- **When Used**: Only when you actively use the voice input feature
- **How Used**: Audio is processed entirely on your device
- **Data**: No audio data leaves your device
- **Storage**: No audio recordings are stored

### Privacy Guarantee

- ✅ **No audio data is transmitted** to external servers
- ✅ **No audio recordings are stored** after processing
- ✅ **Microphone is only active** when you use voice input
- ✅ **You control when voice input is used**

---

## Network Usage

### Internet Permission

AI Keyboard requires the `INTERNET` permission only when you explicitly choose to download AI models from a URL.

- **When Used**: Only when you download models from URL
- **How Used**: Direct download of model files
- **Data**: Only model files are downloaded
- **No Tracking**: No analytics or tracking is performed

### Network State Permission

AI Keyboard requires the `ACCESS_NETWORK_STATE` permission to check network availability before model downloads.

- **When Used**: Before model downloads
- **How Used**: Check network availability
- **Data**: No data is transmitted

---

## Storage Access

### File Access Permission

AI Keyboard requires storage access only when you choose to install models from local files.

- **When Used**: Only when you install models from files
- **How Used**: Read model files from your device
- **Scope**: Only model files are accessed
- **Privacy**: No other files are accessed

### Storage Location

Models are stored in the app's private directory:

- **Location**: `filesDir/models/`
- **Access**: Only accessible to AI Keyboard
- **Privacy**: Other apps cannot access this directory
- **User Control**: You can delete models at any time

## Children's Privacy

AI Keyboard does not knowingly collect any information from children under 13. Since we do not collect any data, this is not applicable.

## Data Security

### Local Storage Security

- **Encrypted Storage**: App data is stored in the app's private directory
- **Access Control**: Other apps cannot access AI Keyboard data
- **Secure Storage**: Models are stored securely in app-private storage

### Model Security

- **SHA-256 Verification**: Models are verified using checksums
- **Secure Downloads**: Model downloads use secure connections (HTTPS)
- **User Control**: You control which models are installed

## Your Rights

### Data Control

- **Delete Data**: You can delete clipboard history, dictionary, and models at any time
- **Uninstall**: Uninstalling the app deletes all local data
- **Model Management**: You can install or remove models at any time

### Privacy Settings

- **Voice Input**: You can disable voice input at any time
- **Clipboard**: You can disable clipboard manager at any time
- **Permissions**: You can revoke permissions in Android settings
