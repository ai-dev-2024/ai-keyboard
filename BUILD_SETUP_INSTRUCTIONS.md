## ‚ö†Ô∏è Required: Android SDK Configuration

The build requires the Android SDK to be configured. You need to create a `local.properties` file.

### Option 1: If Android Studio is Installed

1. Open Android Studio
2. Go to **File ‚Üí Settings ‚Üí Appearance & Behavior ‚Üí System Settings ‚Üí Android SDK**
3. Copy the **Android SDK Location** path
4. Create `local.properties` in the project root:

```properties
sdk.dir=C\:\\Users\\YourUsername\\AppData\\Local\\Android\\Sdk
```

**Important:** Replace the path with your actual SDK location and escape backslashes with double backslashes.

### Option 2: If SDK is Installed Manually

Find your Android SDK installation and create `local.properties`:

```properties
sdk.dir=C\:\\path\\to\\your\\android\\sdk
```

### Option 3: Install Android SDK

If you don't have the Android SDK:

1. **Install Android Studio** (recommended):
   - Download from: https://developer.android.com/studio
   - During installation, it will set up the SDK automatically
   - Default location: `C:\Users\YourUsername\AppData\Local\Android\Sdk`

2. **Or use Command Line Tools**:
   - Download from: https://developer.android.com/studio#command-tools
   - Extract to a folder (e.g., `C:\Android\Sdk`)
   - Run SDK Manager to install required packages

### Required SDK Components

Make sure you have installed:
- Android SDK Platform 34 (or the version specified in `compileSdk`)
- Android SDK Build-Tools
- Android SDK Platform-Tools
- Android Emulator (optional, for testing)

### Verify Setup

After creating `local.properties`, verify the build works:

```bash
.\gradlew.bat clean
.\gradlew.bat assembleDebug
```

## üöÄ Once SDK is Configured

After creating `local.properties`, you can build:

### Debug APK
```bash
.\gradlew.bat assembleDebug
```
**Output:** `app/build/outputs/apk/debug/app-debug.apk`

### Release APK (Unsigned)
```bash
.\gradlew.bat assembleRelease
```
**Output:** `app/build/outputs/apk/release/app-release-unsigned.apk`

### AAB Bundle (Unsigned)
```bash
.\gradlew.bat bundleRelease
```
**Output:** `app/build/outputs/bundle/release/app-release.aab`

## üîç Finding Your SDK Location

### Windows
- Default Android Studio location: `%LOCALAPPDATA%\Android\Sdk`
- Or check: `C:\Users\YourUsername\AppData\Local\Android\Sdk`
- Or check environment variable: `%ANDROID_HOME%`

### Mac
- Default: `~/Library/Android/sdk`
- Or: `/Users/YourUsername/Library/Android/sdk`

### Linux
- Default: `~/Android/Sdk`
- Or: `/home/YourUsername/Android/Sdk`
